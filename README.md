# AndroidApp1 - Tap Counter

This repository contains a simple Android app (Tap Counter) that demonstrates basic Android development concepts.

Features:
- Increment a counter with a "Tap" button
- Reset the counter
- Persist the counter across app restarts using SharedPreferences

Build & Run
1. Open the project in Android Studio.
2. Let Gradle sync and build the project.
3. Run the `app` module on an emulator or device.

Notes
- This project is intentionally minimal for learning purposes.
